Windows & Linux GhostGate

Real-Time DDoS Detection & Automated Firewall Defense

Project Overview

GhostGate is a real-time DDoS detection and mitigation system developed for Windows and Linux environments. It monitors live network traffic and automatically blocks malicious IPs using firewall integration.

Core Features

• Real-time network traffic monitoring
• Machine Learning anomaly detection (Isolation Forest)
• Automated Windows Firewall rule creation
• Temporary & permanent IP banning
• Secure Flask-based admin dashboard
• Traffic flood simulation tool

Technologies Used

Python • Flask • Scapy • Isolation Forest • Windows Firewall • Linux Networking • HTML • CSS
